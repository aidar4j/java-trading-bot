# java-trading-bot
Simple trading bot algorithms for first price sealed bid auction (FPSBA).

## Algorithms

* `ZeroOnlyBidder`
* `RandomBidder`
* `OpponentPlusOneBidder`
* `WinnerPlusOneOrTwoBidder`
* `AveragePlusOneBidder`
* `MedianPlusOneBidder`

## Future algorithms

* `NeuralNetworkBidder`
* `CurveFitBidder`
* `SequencePredictionBidder`
